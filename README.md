Python Tool to validate quantity of ~100,000 cartons across 300+ stores. For use within the PACSUN team.

Use this program to calculate quantity discrepancies from Microsoft D365 and SCI(Manhattan Integration Framework).

Steps:
1. Run & Download Carton reports from SCI(Manhattan Integration Framework) including all "P" and "0" cartons
2. Run & Download Carton Discrepancy(Open in Excel) from Microsoft D365
3. Install the following packages
4. Rename any "⚠️" lines with proper file path & name

Requirements:
- pip install pandas
- pip install pandas openpyxl
